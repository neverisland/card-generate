# Business Card Designer

[简体中文](./README.md)

## Overview

This is a Vite + React + TypeScript business card generator for editing and exporting landscape or portrait business cards.

You can edit name, role, phone, email, address, social handle, and QR content from the left panel, preview the result in real time on the right, and export the final card as `PNG` or `JPG`.

## Tech Stack

- React 19
- TypeScript
- Vite
- Konva / React Konva
- qrcode

## Install and Run

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build locally

```bash
npm run preview
```

## Usage

1. Enter or update business card content in the left panel.
2. Switch between landscape and portrait layouts from the top bar.
3. Change themes or adjust colors in the visual style section.
4. Review the live card preview on the right.
5. Export the result as `PNG` or `JPG`.
